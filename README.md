An interactive fiction experience by DJ OOR.

Play at [oord.art](https://oord.art).

---

## v1 — April 2026

Three playable characters, four transcendental branches, multi-run progression.

- **Wanderer** — the main path. Four runs through the warehouse, each unlocking new branches (Unity, Truth, Goodness, Beauty).
- **Familiar** — unlocked after Run 3. A non-human perspective. Dynamic rod readouts that detect the player's device, session, and time. Camera fourth-wall break.
- **Builder** — unlocked after Familiar. The quiet one behind the speakers.
- Artifact progression chain across runs (yarn knot → matchbook → polaroid → pale antler → shard of light).
- Gated choices visible but locked until conditions are met.
- Persistent state across runs — the game remembers.

Built as a single-page vanilla JS engine with JSON story nodes. No frameworks, no dependencies, no server required.
